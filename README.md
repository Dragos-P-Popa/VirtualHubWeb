# VirtualHub Net

****Requirements:****
<ul>
<li>Apache2 Server</li>
<li>PHP 7.2</li>
<li>MySQL (With VirtualFlight Database)</li>
</ul>

****Installation:****
<ol>
<li>Unzip in hosting directory</li>
<li>Import MySQL Database (VirtualFlight)</li>
<li>Change MySQL authentication details in .env file</li>
<li>Done!</li>
</ol>
