# VirtualHubWeb
VirtualHub Net source code. Used many different APIs to gather necessary data alongside my own database which I used to store files which were served on the website.
Used the [AVWX API](https://avwx.rest/#:~:text=AVWX%20is%20a%20publicly%2Davailable,clean%20and%20thorough%20as%20possible.) to fetch wether data for airports
across the world and the [Infinite Flight API](https://infiniteflight.com/guide/developer-reference/live-api/overview) to fetch information about ATC. Furthermore, I used
MapBox to create and style my own map, which I populated using data which I mined myself from open source projects. Using this data I was able to create detailed
airport maps with taxiways and gates.
